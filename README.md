# RetailMind AI

RetailMind AI is a Real-Time E-Commerce Analytics Dashboard built using Python, Streamlit and MySQL.

## Features
- Customer Management
- Product Management
- Order Management
- Revenue Analytics
- Sales Forecasting
- Best Selling Products
- Interactive Dashboard
- CSV Report Download

## Tech Stack
- Python
- Streamlit
- MySQL
- Pandas
- Plotly
